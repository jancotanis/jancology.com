New Mozilla applications

June 17, 2004 | Comments (0)

A new FireFox 0.9 and a new Thunderbird 0.7 withing one week! Great new software! I’m now using FireFox instead of IE for my browsing as the tabbed browsing feature is much better to handle then a multiple windowed browser — and don’t forget about all exploits available for IE, so it’s safer too. See here “Why you should dump Internet Explorer”.

For css based software development the DOM inspector is a must have. This saves a lot of time in investigating the — sometimes non standard — behaviour of your browser or code.

ThunderBird has replaced my Mozilla 1.7 as default mail client. Seems to start a bit faster and has more room for extensions then good old Mozilla. Biggest drivers for me are the spam detection based on training data and virus safety. I only whished it had an extension to use Hotmail too!
