Any chances for Coolpix 5700 firmware 1.2?

May 5, 2004 | Comments (0)

The end of march Nikon posted version 1.8 of the Coolpix 5000 firmware. This version also had some enhancements like EXIF 2.2, NEF and Picture Transfer Protocol. This camera has been upgraded a long time ago with the Coolpix 5400 but Nikon still invests in software upgrades for it. I now wonder if Nikon plans a firmware upgrade, version 1.2, for the Coolpix 5700 series of camera’s. Although this camera is also superceeded by another model, the 8700, I think some features would be quite handy. Take for example the live histogram, panorama assist! I wonder if we get this… The chances seem to be minor although not zero.
