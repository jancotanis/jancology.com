ftp.gnu.org compromised

August 14, 2003 | Comments (0)

The GNU ftp server has been compromised. A person got root access and now the following message is seen on the server to tell people: hey, we don’t know what we had for software packages and we don’t know what has been changed so please help us out!

This is a good day for the windows lovers as the site is running with the following linux software: Debian GNU/Linux Apache/1.3.26 with mod_python/2.7.8 Python/2.1.3 .
