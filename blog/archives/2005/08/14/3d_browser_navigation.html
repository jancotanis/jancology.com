3D Browser Navigation

August 14, 2005 | Comments (0)

NTT is developing a 3D webbrowser — called SpaceBrowser. This browser is a commercial browser and will be released in october. It is mainly targetted to kiosk use, museum, photo collections and articles. For this last thing you need to setup special content to make it possible to rank articles in a 3D cube depending its properties. The browser is based on Microsoft Internet Explorer and runs only on XP and Windows 2000. See translated page for more information.

A company called 3B also has a kind of 3D browser. The browser is based on FireFox. Special content is setup by city owners. Here you can walk through a Wolfenstein like maze where all kind of websites are ‘hanging’ on the walls.

Current categories are Airport, Entertainment, Female, Kids, Lifestyle, Shopping and Travel. If you want to get in one of the cities you need to request an advertising place. it is also possible to create your own free city. This can be done offline and the result is a .bbb file. This can be viewed within 3B browser. It must say that although it looks nice, it is not somehing really smart and its the people who nee to build the communities. The 3D is only cosmetic instead of really something usable. It is also quite slow to download the website images.
