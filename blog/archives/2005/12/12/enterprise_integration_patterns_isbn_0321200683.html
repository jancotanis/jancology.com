Enterprise Integration Patterns; ISBN 0321200683

December 12, 2005 | Comments (0)
Designing, Building, and Deploying Messaging Solutions

This book shows and explains large set of patterns used to implement a messaging solution. Some of the patterns are explained with an implementation in Java with the JMS api or .Net C# with MSMQ. This book could best be used as a reference manual as reading the book gets hard after ten different patterns. To finish of al sixty-five patterns will take you some time.

Next to the patterns description, the book contains a consistent visual notation for drawing integration solutions. Every pattern in the book has its own icons visualizing its intent.

Some people think this is the best technical book of 2004. I can’t argue but in 2005 I have read better books. Read this book if you want to be ready for 2007 when SOA reaches its heights!… It probably deserves 5 stars if you are ready to rollout your own messaging solution but if you build upon an existing ESB, most patterns should already available to you and the book can only give you some background infortion…
